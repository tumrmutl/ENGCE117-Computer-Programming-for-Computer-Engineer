## โจทย์
จงเขียนโปรแกรมแสดงคำว่า Hello World จำนวน n บรรทัด (โดย n คือตัวเลขที่รับมาจากผู้ใช้)

## FIX CODE
```c++
#include <stdio.h>
```

## TEST CASE
### Input
```bash
5
```
### Output
```bash
[1] Hello world
[2] Hello world
[3] Hello world
[4] Hello world
[5] Hello world
```

## TEST CASE
### Input
```bash
3
```
### Output
```bash
[1] Hello world
[2] Hello world
[3] Hello world
```
