## โจทย์
จงเขียนคำสั่งในการสร้างและเรียกใช้งานฟังก์ชัน GetSet() เพื่อรับจำนวนสมาชิกและค่าของแต่ละสมาชิกในเซต จากคีย์บอร์ด ตามโปรโตไทป์ต่อไปนี้ 

## Pre-code
```c++
#include <stdio.h>
int *GetSet( int * ) ;

int main() {
   int *data, num ;
   data = GetSet( &num ) ;
   return 0 ;
}//end function
```

## Testcase 1
### Input
```bash
533
```
### Output
```bash
533
```
